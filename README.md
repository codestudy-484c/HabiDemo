# HabiDemo: UI Control Demo

A lightweight, mobile-oriented habit tracking app designed to demo and test various interactive UI control elements of [MockUI Project](https://github.com/codestudy-484c/mockui-project).



# 🚀 Purpose

The primary goal of this application is to serve as a functional sandbox for UI/UX exploration. It showcases how different control elements can be utilized to capture diverse data types—from binary states to granular scales—within a cohesive "Habit Tracker" context.
